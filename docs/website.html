Student Faculty Database

I made this project in my data strucutres and algorithms class using C++.

It uses a templated lazily balanced binary search tree made from scratch.

It is capable of storing a student's ID number, name, year, major, and advisor ID.

It is also capable of storing a faculty member's ID number, name, position, department,

and advisee ID number.

When ran the following options are provided:

1. Print all students and their information

2. Print all students and their information

3. Find and display student information given a student ID

4. Find and display faculty information given a faculty ID

5. Add a new student

6. Delete a student given their ID number

7. Add a new faculty member

8. Delete a faculty member given their ID number

9. Change a students advisor given the student ID and new faculty ID

10. Change a faculty advisee given the faculty ID number and new student ID

11. Exit
